1. Úvod

Expo poskytuje jednoduchý způsob, jak spustit a otestovat React Native aplikace přímo v prohlížeči pomocí nástroje Snack.

2. Co je Snack?

Snack je online editor od týmu Expo, který vám umožňuje psát a testovat React Native kód okamžitě.

3. Jak začít

  1. Otevřete Snack na Expo.
  2. Klikněte na "Create a Snack" a začněte psát kód v editoru.
  3. Sledujte simulaci aplikace na pravé straně obrazovky.

4. Příklad aplikace

import React from 'react';
import { Text, View } from 'react-native';

export default function App() {
  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Text>Ahoj, světe! Toto je moje první aplikace v Expo Snacku!</Text>
    </View>
  );
}

5. Uložení a sdílení

Chcete-li uložit projekt, přihlaste se na Snack pomocí Google nebo GitHub. Váš projekt můžete sdílet pomocí vygenerovaného odkazu.
